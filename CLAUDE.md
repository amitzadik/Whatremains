# הוראות לסוכן

## Git workflow (חוק מהמשתמש)
- אחרי כל שינוי בקוד — תמיד הרץ:
  ```
  git add . && git commit -m "[מה השתנה]" && git push origin main
  ```
- אל תשאל את המשתמש אם לעשות push — פשוט תעשה.
- כל הקומיטים הולכים ישירות ל-`main`, לא לסניפים אחרים.
- אם push נכשל — דווח על השגיאה ופרט מה הסיבה.
